
start() — запускает новый поток, вызывая его метод run().
Поток начинает выполняться независимо от текущего (главного) потока.

run() — содержит код, который будет выполнен в потоке.
Вызывается автоматически после вызова start().
Если вызвать его напрямую, поток не будет создан — код просто выполнится в текущем потоке.

setName() - изменяет имя объекта Thread или задаёт имя потока, полезно для идентификации при отладке.

setPriority(int priority) Устанавливает приоритет для потока 1-10
-Thread.MIN_PRIORITY (1)
-Thread.NORM_PRIORITY (5, по умолчанию)
-Thread.MAX_PRIORITY (10)

interrupt() - прерывает этот поток(посылает сигнал на прерывание потока.
Сам поток должен регулярно проверять, был ли он прерван, через метод isInterrupted())

isAlive() - возвращает true, если поток был запущен и ещё не завершил свою работу.

join(long millis) — приостанавливает текущий поток до завершения указанного потока или
до истечения времени (в миллисекундах). Без аргументов ждёт завершения потока бесконечно.
